# Dose Calculator

A simple weight-based dose calculator built using HTML, CSS, and JavaScript.

## Live Demo
https://raveena0805.github.io/DOSE---CALCULATOR/

## Purpose
To help ensure that drugs are administered in the correct dose based on body weight.

## Features
- Weight-based dose calculation
- Target audience: healthcare professionals including doctors, pharmacists, nurses, and patients
- Clean and simple user interface
- Beginner-friendly project with scope for further enhancements
